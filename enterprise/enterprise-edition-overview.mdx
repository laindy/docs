At **Hymalaia**, we aim to make our open-source offering as powerful as possible. However, large enterprises need additional support, which is why we offer enterprise-oriented features and functionality.

## EE Features

### Security and Access Control
- **Single Sign-On (SSO)**: Integrate OIDC + SAML for streamlined authentication.
- **Role-Based Access Control (RBAC)**: Manage user permissions efficiently.
- **Credential Encryption**: Ensure robust security for sensitive data.
- **Document Permission Inheritance**: Simplify permission management.
- **API Key Authentication**: Secure API integrations with dedicated keys.

### Enhanced Functionality
- **Additional APIs**: Access extended API endpoints.
- **White Labeling**: Customize the interface to match your brand.
- **Admin Dashboard / Analytics**: Gain insights into usage and performance.
- **Query History**: Track and analyze queries.
- **Usage Limits Configuration**: Set and manage usage limits.

### Support and Services
- **Enterprise-Level Support**: Benefit from priority support.

## Roadmap
- **Expert Suggestion**: AI-powered recommendations.
- **Personalized Search**: Tailored search results.
- **Org-Structure Importing**: Integrate your org's hierarchy.
